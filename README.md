# SOLAR-TRACKING-SYSTEM
"Designed and developed as part of Embedded & IoT project to demonstrate practical implementation of renewable energy optimization."
Project Overview

This project implements a Solar Tracking System using ESP32 that automatically aligns a solar panel towards maximum sunlight direction to improve energy efficiency.
The system uses LDR sensors to detect light intensity and a servo motor to adjust panel orientation. Real-time data monitoring is integrated using IoT (Blynk platform).

Objective
Maximize solar energy absorption
Improve efficiency compared to fixed solar panels
Demonstrate IoT-based remote monitoring

🛠️ Hardware Components
ESP32
LDR Sensors (4x for light detection)
Servo Motor (SG90 / MG995)
DHT11 (Temperature & Humidity Sensor)
LED Matrix Display
Solar Panel (Mini Prototype)
Power Supply
